## 你好，世界！

欢迎来到我的博客。这是使用 SvelteKit 构建的现代博客系统，支持 Markdown 写作。

### 功能特性

- **静态生成** - 所有页面在构建时预渲染为静态 HTML
- **Markdown 支持** - 使用 mdsvex 编译 Markdown 为 Svelte 组件
- **响应式设计** - 适配桌面和移动端
- **暗色模式** - 支持亮色/暗色/跟随系统三种主题

```typescript
const greeting = 'Hello, World!';
console.log(greeting);
```
